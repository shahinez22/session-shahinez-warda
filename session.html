ما هي الـ Session؟

بروتوكول HTTP بطبيعته stateless — أي أن كل طلب مستقل تمامًا ولا يتذكر الخادم من أنت. الـ Session هي الحل: تُخزِّن بيانات المستخدم على الخادم وتعطيه معرّفًا فريدًا (PHPSESSID) يُرسَل عبر Cookie للمتصفح.

🖥️

التخزين على الخادم

البيانات تُحفظ في ملفات مؤقتة على الخادم، لا في المتصفح — أكثر أمانًا من Cookies.

🔑

معرّف فريد

كل مستخدم يحصل على session_id عشوائي طويل يربطه ببياناته المحفوظة.

⏱️

مؤقتة بطبيعتها

تنتهي عند إغلاق المتصفح أو انتهاء المهلة المحددة من جانب الخادم.

🌐

تعمل عبر الصفحات

تتشارك البيانات بين جميع صفحات الموقع طوال فترة تسجيل الدخول.

// 02

كيف تعمل خطوة بخطوة

فهم دورة حياة الـ Session يساعدك على استخدامها بشكل صحيح وتجنّب المشاكل الشائعة.

01

استدعاء session_start()

PHP تتحقق من وجود Cookie بالاسم PHPSESSID. إن لم تجد، تُنشئ session_id جديدًا.

02

إرسال الـ Cookie للمتصفح

PHP ترسل هيدر Set-Cookie تلقائيًا يحمل الـ PHPSESSID — المتصفح يحفظه.

03

تخزين البيانات في $_SESSION

تُكتب البيانات في ملف مؤقت على الخادم مرتبط بالـ session_id.

04

الطلبات اللاحقة

المتصفح يُرسل الـ PHPSESSID مع كل طلب، وPHP تُحمّل البيانات المناسبة تلقائيًا.

05

إنهاء الجلسة

عبر session_destroy() أو انتهاء المهلة أو إغلاق المتصفح.

// 03

الكود الأساسي

بدء الجلسة وحفظ بيانات

index.php 
<?php
// يجب أن تكون أول شيء في الملف قبل أي إخراج
session_start();

// تخزين البيانات
$_SESSION['username']   = 'Ahmed';
$_SESSION['role']       = 'admin';
$_SESSION['login_time'] = time();

echo "Session ID: " . session_id();
?>

قراءة البيانات في صفحة أخرى

dashboard.php 
<?php
session_start();

// حماية الصفحة من غير المسجّلين
if (!isset($_SESSION['username'])) {
    header("Location: login.php");
    exit;
}

$name = htmlspecialchars($_SESSION['username']);
echo "مرحباً {$name}!";
?>

تسجيل الخروج بشكل آمن

logout.php 
<?php
session_start();

// 1. حذف جميع متغيرات الجلسة
$_SESSION = [];

// 2. حذف Cookie من المتصفح
if (ini_get("session.use_cookies")) {
    $p = session_get_cookie_params();
    setcookie(session_name(), '', time() - 3600,
        $p['path'], $p['domain'],
        $p['secure'], $p['httponly']
    );
}

// 3. تدمير الجلسة على الخادم
session_destroy();

header("Location: login.php");
exit;
?>
// 04

مثال متقدم — سلة التسوق

استخدام الـ Session لبناء سلة تسوق متكاملة تعمل عبر جميع صفحات المتجر.

cart.php 
<?php
session_start();

// تهيئة السلة
if (!isset($_SESSION['cart'])) {
    $_SESSION['cart'] = [];
}

function addToCart($id, $name, $price) {
    if (isset($_SESSION['cart'][$id])) {
        $_SESSION['cart'][$id]['qty']++;
    } else {
        $_SESSION['cart'][$id] = [
            'name'  => $name,
            'price' => $price,
            'qty'   => 1,
        ];
    }
}

function getTotal() {
    $total = 0;
    foreach ($_SESSION['cart'] as $item) {
        $total += $item['price'] * $item['qty'];
    }
    return $total;
}

// استخدام
addToCart(1, 'كتاب PHP', 25.00);
addToCart(2, 'دورة Laravel', 99.00);
echo "الإجمالي: $" . getTotal();
?>
// 05

الأمان أولًا

الـ Session غير الآمنة هي بوابة للاختراق. هذه أهم الإعدادات التي يجب تطبيقها.

✓
HttpOnly Cookie

يمنع JavaScript من الوصول إلى الـ Cookie — يحمي من XSS.

✓
Secure Cookie

ترسل الـ Cookie فقط عبر HTTPS — مطلوب في بيئة الإنتاج.

✓
session_regenerate_id()

جدّد معرّف الجلسة بعد تسجيل الدخول — يمنع Session Fixation.

✓
انتهاء صلاحية تلقائي

أنهِ الجلسة تلقائيًا بعد فترة خمول (30 دقيقة مثلًا).

✓
strict_mode

يرفض session_id غير المُنشأة من طرف الخادم.

secure_session.php 
<?php
// الإعدادات قبل session_start()
ini_set('session.cookie_httponly', 1);
ini_set('session.cookie_secure',   1);
ini_set('session.use_strict_mode', 1);
ini_set('session.cookie_samesite', 'Lax');

session_start();

// تجديد المعرّف بعد تسجيل الدخول
session_regenerate_id(true);

// انتهاء صلاحية بعد 30 دقيقة
$timeout = 1800;
if (isset($_SESSION['last_activity']) &&
    (time() - $_SESSION['last_activity']) > $timeout) {
    session_destroy();
    header("Location: login.php?expired=1");
    exit;
}
$_SESSION['last_activity'] = time();
?>
// 06

مرجع الدوال

جميع دوال الـ Session في PHP مع شرح موجز لكل منها.

الدالة الوصف
session_start() بدء جلسة جديدة أو استئناف جلسة موجودة
session_id() الحصول على معرّف الجلسة الحالية أو تعيينه
session_name() اسم الـ Cookie (افتراضي: PHPSESSID)
session_regenerate_id(true) تجديد المعرّف وحذف الجلسة القديمة (للأمان)
session_unset() حذف جميع متغيرات $_SESSION
session_destroy() حذف ملف الجلسة من الخادم نهائيًا
session_get_cookie_params() استرجاع إعدادات الـ Cookie الخاصة بالجلسة
session_status() حالة الجلسة: PHP_SESSION_NONE / ACTIVE / DISABLED
